Width

Tag utilizada para definir a largura do elemento na tela e pode ter dinifições de limite tanto para largura máxima "max-width" quando para largura mínima "min-width"

Max Width

Nessa propriedade é possível definir a largura máxima, caso sua tela fique abaixo da dimensão máxima a tag irá se adapatar, porém se a tela ultrapassar a tag respeitar o limite.
