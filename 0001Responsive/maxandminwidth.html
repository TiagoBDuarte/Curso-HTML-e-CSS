Max and Min Width

Na div 1000px a borda só de adapta quando atinge 1000px para cima, quando a tela diminiu abaixo de 1000px continua ultrapassar

.ex1 {
border: 1px solid red;
min-width: 1000px;
}

Na div 700px a borda só de adapta quando atinge 700px para baixo, quando a tela diminiu abaixo de 700px não ultrapassa

.ex2 {
border: 1px solid blue;
max-width: 700px;
}
